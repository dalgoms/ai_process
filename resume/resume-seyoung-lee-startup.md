# 이세영

**AI 활용 마케팅 시스템을 직접 설계하고 구축하는 Growth Marketer**

---

## Summary

마케팅 실무 경험을 기반으로, 반복적인 업무를 자동화 파이프라인으로 바꾸는 일을 해왔습니다.
SEO, 퍼포먼스 마케팅, 콘텐츠 전략으로 리드 유입 구조를 만들고, GPT·Make·Notion을 연결해 콘텐츠 생산부터 리드 관리까지 자동화하는 시스템을 직접 구축·운영하고 있습니다.
마케팅과 제품 경험의 경계에서, AI를 도구로 활용해 실질적인 비즈니스 성과를 만드는 데 집중합니다.

## Key Achievements

**리드 유입 채널 구축**
서비스 문의가 분산되어 있고 유입 경로가 불명확한 상태에서, SEO 전략과 콘텐츠 구조를 재설계하여 검색 기반 리드 유입 채널을 구축했습니다.
→ 안정적인 문의 유입 구조 확보 및 전환율 개선

**마케팅 자동화 파이프라인 구축**
뉴스 수집, 콘텐츠 리서치, 초안 작성에 매일 반복적인 시간이 소요되는 문제를 해결하기 위해 Notion + Make + GPT 기반 자동화 파이프라인을 구축했습니다.
→ 콘텐츠 리서치 시간 약 80% 절감

**리드 캡처 및 팔로업 자동화**
웹사이트 문의 → Notion CRM → 알림 → 미응대 리드 자동 리마인더까지 이어지는 리드 관리 시스템을 설계했습니다.
→ 리드 누락 방지, 응대 속도 개선

**웹사이트 구조 분석 도구 개발**
경쟁사 및 파트너사 웹사이트의 SEO 구조를 빠르게 파악하기 위해 WebScout을 직접 개발했습니다.
→ 사이트맵 분석, SEO 신호 추출, UX 인사이트 자동 생성

**헬스케어 AI MVP 제안서 설계**
헬스케어 앱의 낮은 리텐션 문제를 분석하고, AI 기반 건강 동반자 컨셉(AI 건강이)의 MVP 제안서를 작성했습니다.
→ 서비스 구조 분석, 기능 정의, CRM 연동 설계, UI 와이어프레임 포함

## Projects

**WebScout**
웹사이트 구조를 크롤링하여 SEO 신호와 UX 인사이트를 자동 분석하는 도구

- URL 입력만으로 사이트맵 구조, navigation depth, 콘텐츠 클러스터를 분석
- 경쟁사 분석과 서비스 구조 파악에 활용
- Next.js, Vercel, Make.com

**AI 건강이 (AI Health Companion)**
헬스케어 앱의 리텐션 문제를 해결하기 위한 AI 건강 알림 MVP 제안서

- 아침/저녁 건강 루프, 건강 타임라인, 챗봇 기능 설계
- 나만의닥터 서비스 구조 분석을 통한 시장 인사이트 도출
- Braze CRM 연동 시나리오 설계

**Marketing Automation System**
콘텐츠 생산, 리드 관리, 알림을 자동화하는 마케팅 운영 시스템

- Notion Work Inbox → GitHub Issue → GPT → 배포까지 엔드투엔드 파이프라인
- 문의폼 → CRM → Telegram/Email 멀티채널 알림 자동화
- Make.com, Notion API, GitHub Actions
