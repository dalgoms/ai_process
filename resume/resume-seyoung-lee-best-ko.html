WebScout — 서비스 구조 분석 도구 live github

웹사이트 아키텍처 평가, 숨은 페이지 탐지, SEO 콘텐츠 클러스터 매핑을 위한 내부 분석 도구입니다. 경쟁사 구조 파악과 유입 기회 식별을 위한 실무 목적으로 활용했습니다.

기술 스택: Next.js / TypeScript / Vercel

Automation Process Playbook github

업무 자동화를 위한 중앙 운영 구조를 설계한 프로젝트입니다. Notion 작업 등록 → GitHub Issue → 코드 작업 → PR → CI → 배포로 이어지는 흐름을 정리하고, 재사용 가능한 운영 문서와 자동화 플레이북을 구축했습니다.

Business Automation System

현재 운영 중인 실무 시스템입니다. 문의폼 → Make → Notion CRM → Telegram/Email로 이어지는 리드 캡처 구조, 스케줄 기반 팔로업 자동화, 콘텐츠 초안 보조, Telegram·Email·Slack·KakaoTalk·SMS 멀티채널 알림 연동을 포함합니다.

MEFIMAKE — Meta 광고 소재 생성기 live github

퍼포먼스 마케팅 팀의 크리에이티브 제작 병목을 해결하기 위해 만든 브라우저 기반 광고 소재 IDE입니다. AI 카피 생성(헤드라인·서브텍스트·CTA), 멀티사이즈 자동 내보내기(1:1, 16:9, 9:16), 배치 생성(5개 변형 × 3사이즈 = 15장)을 하나의 인터페이스에서 처리합니다. WebScout·Meta Library 연동으로 경쟁 소재 리서치부터 제작까지 연결됩니다.

기술 스택: Vanilla JS / CSS3 / Vercel

알림 자동화 (구글/네이버)

구글 알리미와 네이버 키워드 알림을 활용한 업계·경쟁사·키워드 모니터링 자동화 구조입니다. 수집된 알림을 Make를 통해 Notion DB로 자동 정리하고, 주요 변화 발생 시 Telegram/Slack으로 실시간 알림을 전달합니다. 마케팅 리서치와 경쟁 분석의 초기 탐지를 자동화하여 대응 속도를 높였습니다.
