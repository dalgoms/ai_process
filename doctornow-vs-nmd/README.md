# 비대면 진료 플랫폼 비교 분석 — 닥터나우 vs 나만의닥터

> 비대면 진료 시장의 양대 플랫폼을 구조적으로 비교하면서 정리한 내용입니다.

## 배경

비대면 진료 시범사업이 본격화되면서, 닥터나우와 나만의닥터가 시장을 양분하고 있습니다. 두 서비스의 웹사이트 구조, SEO 전략, 전환 설계를 비교하여 각각의 강점과 성장 기회를 정리했습니다.

## 서비스 요약

| 항목 | 닥터나우 | 나만의닥터 |
|---|---|---|
| URL | [doctornow.co.kr](https://doctornow.co.kr) | [my-doctor.io](https://my-doctor.io) |
| 포지셔닝 | "대한민국 1위 비대면 진료 앱" | "대한민국 1위 비대면 진료·약 배달 어플" |
| 핵심 서비스 | 비대면 진료, 병원 찾기, 약국 찾기, 건강 매거진, 실시간 상담 | 비대면 진료, 약 배달, 닥터포인트, 닥터캐시(앱테크), 최저가 병원 |
| 사이트맵 URL 수 | **31 pages** | **19,934 pages** |
| 차별화 | 앱 UX·사회적 증거 (25만+ 후기, 4.8점) | 대규모 콘텐츠 SEO (진료비 비교 15K+ 페이지) + 리텐션 (포인트, 캐시) |
| 웹 전략 | 앱 중심, 웹은 간결한 게이트웨이 | 콘텐츠 SEO 머신 + 앱테크 리텐션 |
| 수익 모델 | 진료비 + 병원 제휴 | 진료비 + 병원/약국 제휴 + 포인트 생태계 |
