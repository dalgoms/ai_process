# 비대면 진료 플랫폼 비교 분석 — 닥터나우 vs 나만의닥터

> 비대면 진료 시장의 양대 플랫폼인 **닥터나우**와 **나만의닥터**를 웹 구조와 성장 전략 관점에서 비교 분석한 리서치입니다.

---

##

닥터나우와 나만의닥터는 동일한 비대면 진료 시장에 있지만 **획득 전략과 제품 구조가 구조적으로 다르게 설계된 플랫폼**입니다.

- **DoctorNow → Transaction Platform (앱 실행 중심)**
- **나만의닥터 → Content Acquisition Platform (SEO 기반 유입)**

웹 구조 기준 페이지 규모

- **DoctorNow: 31 pages**
- **나만의닥터: 19,934 pages**

이 분석은 공개 웹 구조 데이터를 기반으로 두 플랫폼의 **제품 전략, SEO 구조, 성장 기회**를 비교 정리한 리서치입니다.

---

## 배경

비대면 진료 시범사업이 본격화되면서 **닥터나우**와 **나만의닥터**가 국내 비대면 진료 시장을 대표하는 플랫폼으로 자리잡고 있습니다.

본 문서는 두 서비스의 **웹사이트 구조, SEO 전략, 전환 설계**를 중심으로 비교하여 각 플랫폼의 **성장 전략과 구조적 차이**를 정리했습니다.

## Key Insight

두 서비스의 가장 큰 차이는 **유입 전략과 제품 구조**입니다.

### DoctorNow

- 앱 중심 실행 플랫폼
- 웹은 전환 게이트웨이 역할

### 나만의닥터

- 콘텐츠 기반 SEO 유입 플랫폼
- 콘텐츠 → 진료 전환 구조

즉,
DoctorNow → Transaction Platform
나만의닥터 → Content Acquisition Platform

## Analysis Tool

본 분석은 직접 제작한 **WebScout** 기반으로 진행되었습니다.

WebScout는 웹사이트 구조를 크롤링하여 다음 데이터를 수집하고 분석하는 도구입니다.

- 사이트맵 기반 페이지 수집
- 웹 구조 및 콘텐츠 카테고리 분석
- 페이지 depth 구조 분석
- SEO 콘텐츠 규모 분석

수집된 웹 구조 데이터를 기반으로 **GPT-4o를 활용해 구조적 인사이트를 도출했습니다.**

## Methodology

본 분석은 **내부 데이터가 아닌 공개 웹 구조 데이터를 기반으로 진행되었습니다.**

분석 과정

1. 공개 웹사이트 구조 크롤링
2. 사이트맵 기반 URL 규모 분석
3. 콘텐츠 카테고리 구조 분석
4. SEO 및 전환 구조 비교
5. AI 기반 전략 인사이트 도출
