# 성장 기회 정의 — 닥터나우 vs 나만의닥터

> 각 플랫폼이 가진 구조적 기회를 Growth Marketing 관점에서 정리

---

## 닥터나우 — 기회 영역

### 기회 1: 리텐션 메커니즘 부재

**현상:** 콘텐츠로 유입은 강하지만, 비진료 시점에 사용자를 붙잡는 장치가 약함

| 지표 | 현재 추정 | 기회 |
|---|---|---|
| 비진료 시점 DAU | 콘텐츠 소비 위주 | 건강관리 루틴 + 리워드로 DAU 증가 |
| 재방문율 | 증상 발생 시에만 | 일상적 건강 데이터 기록 유도 |
| LTV | 진료 건당 과금 | 구독/포인트로 LTV 연장 |

**나만의닥터는 이미 닥터캐시/닥터포인트로 이 문제를 해결 중**

---

### 기회 2: 가격 투명성 기능

**현상:** 닥터나우는 "좋은 의사 선택"에 초점. 가격 비교 기능은 미흡

| 사용자 니즈 | 닥터나우 | 나만의닥터 |
|---|---|---|
| "탈모약 어디가 싸?" | 의사 리뷰 기반 | 최저가 병원 예약 |
| "비급여 진료비 비교" | X | O |
| 가격 검색 SEO | X | 위고비, 가다실 등 키워드 |

---

### 기회 3: B2B 제휴 확장

**현상:** 병원/약국 제휴 페이지는 있으나 B2B 마케팅 채널로서의 웹 존재감이 약함. 나만의닥터는 "의사/약사 선생님이신가요?" 섹션을 홈에 직접 노출하며 병원 신환 유치를 적극 소구.

---

## 나만의닥터 — 기회 영역

### 기회 1: SEO 콘텐츠 자산 빈약

**현상:** 검색 유입 → 앱 설치 퍼널이 거의 없음

| 지표 | 현재 추정 | 기회 |
|---|---|---|
| 오가닉 검색 유입 | 낮음 (브랜드 검색 위주) | 질환/증상별 콘텐츠 → 자연 유입 |
| 콘텐츠 페이지 수 | 건강연구소 수십 건 | 수백~수천 건 확대 가능 |
| 키워드 커버리지 | 서비스명 위주 | 롱테일 건강 키워드 |

**닥터나우의 건강 매거진, 질환백과, 의약품사전 규모가 압도적 우위**

---

### 기회 2: 웹 전환 구조 약함

**현상:** 웹사이트가 사실상 앱 다운로드 랜딩 페이지 역할만 수행

- 진료과별 전용 랜딩 페이지 없음
- 증상별 검색 결과 페이지 없음
- 웹에서 직접 비대면 진료 시작 불가 (앱 필수)

**시사점:** 모바일 웹 → 앱 전환율을 높이려면 웹에서도 가치 제공이 필요. "앱 다운로드" CTA만으로는 전환 장벽이 높음.

---

### 기회 3: UGC/커뮤니티 부재

**현상:** 사용자 후기, 실시간 질문 등 참여형 콘텐츠가 없음

| 항목 | 닥터나우 | 나만의닥터 |
|---|---|---|
| 실시간 Q&A | O | X |
| 사용 후기 노출 | 250,000+ 캐러셀 | X (웹에서 미노출) |
| 커뮤니티 | 실시간 의료상담 | X |

닥터나우는 실시간 의료상담이 SEO + 신뢰 구축 + 전환의 삼중 역할 수행

---

## 양사 공통 기회

### 1. 구조화 데이터 (Schema.org)

두 사이트 모두 MedicalOrganization, FAQPage, MedicalWebPage 등의 구조화 데이터 적용 여지가 큼. Google의 건강 정보 SERP 노출에 직결되는 요소.

### 2. 비대면 진료 서비스 웹앱화

현재 두 서비스 모두 "앱 다운로드 → 진료"라는 퍼널인데, PWA 또는 웹앱으로 웹에서 직접 진료 접수까지 가능하게 하면 전환율 대폭 개선 가능.

### 3. 퍼포먼스 마케팅 랜딩 최적화

광고 캠페인(Google Ads, Meta 등)의 도착 페이지로 사용할 증상별/진료과별 전용 랜딩 페이지가 부족. A/B 테스트 가능한 랜딩 구조 필요.
