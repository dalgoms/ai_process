## 콘텐츠 전략 비교

### 닥터나우 — "콘텐츠 SEO 머신"

```
홈 → 건강 정보 → 건강 매거진 / 1분 FAQ / 의약품 사전 / 질환백과
홈 → 비대면 진료 → 소아과 / 감기 / 탈모 / 여드름 / ...
홈 → 실시간 의료 상담 → 질문 / 답변
```

**강점:**
- 건강 매거진, 질환백과, 의약품 사전 등 SEO 콘텐츠 자산이 방대
- "탈모 원인", "감기 증상" 같은 롱테일 키워드로 자연 유입 확보
- 실시간 의료 상담 → UGC(User Generated Content) 축적
- 진료과별 전용 랜딩 (소아과, 탈모, 여드름 등)

**구조적 특징:**
- 콘텐츠 → 진료 전환 퍼널이 명확 (매거진 읽기 → 비대면 진료 시작)
- FAQ 섹션이 홈페이지에 직접 노출 → 전환 장벽 제거
- 사용 후기 캐러셀 (250,000+ 후기) → 사회적 증거

### 나만의닥터 — "리텐션 엔진"

```
홈 → 비대면 진료
홈 → 닥터포인트 (진료비 할인)
홈 → 닥터캐시 (건강관리 리워드)
홈 → 최저가 병원 예약
홈 → 일요일/야간 병원 찾기
홈 → 건강연구소 (콘텐츠)
```

**강점:**
- 포인트/캐시 시스템으로 사용자 락인
- "건강관리만 해도 리워드" → 비진료 시점에도 앱 사용 유도
- 최저가 비교 → 가격 투명성으로 신뢰 구축
- 일요일/야간 병원 찾기 → 긴급 수요 캡처

**구조적 특징:**
- 서비스 기능 중심 (진료, 포인트, 캐시, 병원찾기)
- 콘텐츠 자산은 상대적으로 얕음 (건강연구소 규모 작음)
- 단일 랜딩 구조 → 스크롤 기반 정보 전달

## SEO 구조 비교

| 항목 | 닥터나우 | 나만의닥터 |
|---|---|---|
| 콘텐츠 볼륨 | 대량 (매거진, FAQ, 사전, 백과) | 소량 (건강연구소) |
| 키워드 커버리지 | 넓음 (질환별, 증상별, 약물별) | 좁음 (서비스 기능 중심) |
| 진료과별 랜딩 | O (소아과, 탈모, 여드름 등) | X (통합 랜딩) |
| UGC | 실시간 의료상담 (Q&A) | X |
| 구조화 데이터 | 미확인 | 미확인 |
| 블로그/매거진 | O (건강 매거진) | O (건강연구소, 규모 작음) |

**시사점:**

닥터나우는 "건강 정보 검색 → 서비스 전환"이라는 SEO 퍼널을 체계적으로 구축한 반면, 나만의닥터는 앱 내 리텐션에 집중하고 웹 SEO 유입은 상대적으로 약함.

## 전환 구조 비교

| 단계 | 닥터나우 | 나만의닥터 |
|---|---|---|
| 인지 | SEO 콘텐츠, 매거진 | 브랜드 광고, 앱스토어 |
| 관심 | 증상별 상세 페이지, 후기 | 서비스 설명, 가격 비교 |
| 전환 | "비대면 진료 시작하기" CTA | "앱 다운로드" CTA |
| 리텐션 | 콘텐츠 재방문, 실시간 상담 | 포인트, 캐시, 건강관리 |

**핵심 차이:**

- 닥터나우: **웹 → 앱** 전환 (콘텐츠로 유입 → 앱 설치 유도)
- 나만의닥터: **앱 중심** (웹은 앱 다운로드 게이트웨이)
