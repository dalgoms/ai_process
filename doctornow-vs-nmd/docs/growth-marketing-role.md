# Growth Marketing 방향 — 비대면 진료 플랫폼

> 기여할 수 있는 영역 정리

---

## 핵심 기여 영역

```
유입 설계 (SEO/콘텐츠)  ·  전환 최적화 (CTA/랜딩)  ·  리텐션 구조 (리워드/UGC)
```

## 닥터나우에서의 기여 방향

| 영역 | 현재 상태 | 기여 가능 |
|---|---|---|
| 유입 | 앱 광고 중심, 웹 SEO 약함 (31p) | 앱 내 콘텐츠 웹 퍼블리싱 → SEO 유입 10배 확대 |
| 전환 | 전환 페이지 depth 깊음, 랜딩 부재 | 증상별/진료과별 전용 랜딩 + A/B 테스트 |
| 리텐션 | 앱 내 콘텐츠, Q&A | 건강관리 루틴 + 리워드 시스템 구축 |

## 나만의닥터에서의 기여 방향

| 영역 | 현재 상태 | 기여 가능 |
|---|---|---|
| 유입 | SEO 콘텐츠 대량 (19,934p), depth 깊음 | 카테고리 허브 구축 → 접근성 개선 → 인덱싱 확대 |
| 전환 | CTA 약함, 전환 퍼널 불명확 | 콘텐츠→진료 맞춤 CTA + 딥링크 |
| 리텐션 | 포인트/캐시 강함 | 후기/리뷰 시스템으로 사회적 증거 보강 |

## 이전 경험에서 적용 가능한 것

| 경험 | 적용 |
|---|---|
| WebScout 개발 (사이트 구조 분석 + AI 진단) | 데이터 기반 SEO 전략 수립 |
| UTM + GA 퍼널 분석 | 채널별 전환 추적 체계 구축 |
| CRM 자동화 (Notion + Make.com) | 리드/환자 관리 자동화 |
| Ad Creative Tool | 퍼포먼스 광고 소재 자동 생성 |

## 성과 목표

| 기간 | 닥터나우 | 나만의닥터 |
|---|---|---|
| 30일 | 콘텐츠 SEO 파이프라인 설계, 상위 50개 키워드 타겟 | depth 4+ 접근성 개선안 설계, 후기 노출 프로토타입 |
| 60일 | 웹 콘텐츠 500+ 페이지 퍼블리싱 | 카테고리 허브 구축, 전환 CTA A/B 테스트 시작 |
| 90일 | SEO 오가닉 유입 3배 증가 | 웹→앱 전환율 2배 개선 |
