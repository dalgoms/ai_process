# 성과 측정 프레임워크 — 비대면 진료 플랫폼

## North Star Metric

| 대상 | 지표 | 정의 | 목표 |
|---|---|---|---|
| 닥터나우 | 웹 경유 앱 설치 수 | SEO 콘텐츠 유입 → 앱 설치 전환 | 월 10,000+ |
| 나만의닥터 | 콘텐츠 경유 첫 진료 수 | 웹 콘텐츠 유입 → 앱 설치 → 첫 진료 완료 | 월 5,000+ |

## 유입 지표

| 지표 | 정의 | 닥터나우 목표 | 나만의닥터 목표 |
|---|---|---|---|
| 오가닉 검색 유입 | 검색엔진 경유 웹 방문 수 | 현재 대비 10배 | 현재 대비 1.5배 (접근성 개선) |
| 인덱싱 페이지 수 | 검색엔진에 인덱싱된 URL 수 | 31 → 2,000+ | 19,934 중 depth 4+ 인덱싱율 개선 |
| 키워드 커버리지 | 검색 결과 1페이지 노출 키워드 수 | 50+ 롱테일 키워드 | 기존 키워드 순위 유지 + 신규 300+ |

## 전환 지표

| 지표 | 정의 | 닥터나우 목표 | 나만의닥터 목표 |
|---|---|---|---|
| 웹→앱 전환율 | 웹 방문 → 앱 설치 비율 | 현재 대비 2배 | 현재 대비 3배 |
| CTA 클릭율 | 콘텐츠 페이지 CTA 클릭 / 페이지 뷰 | 5%+ | 3%+ |
| 랜딩 페이지 전환율 | 전용 랜딩 → 앱 설치 | 15%+ | 10%+ |

## 실패 기준 (Kill Metrics)

| 지표 | 실패 기준 | 의미 |
|---|---|---|
| SEO 유입 변화 없음 | 90일 후 오가닉 유입 변화 < 20% | 콘텐츠 전략 재검토 필요 |
| 전환율 하락 | 랜딩 변경 후 전환율 10% 이상 하락 | 즉시 롤백, A/B 테스트 재설계 |
| depth 개선 효과 없음 | 허브 구축 후 인덱싱율 변화 < 10% | 내부 링크 전략 재검토 |
