# 비대면 진료 시장 환경 조사

---

## 시장 현황

2024년 비대면 진료 시범사업 전면 확대 이후, 시장이 급성장. 누구나 365일 24시간 비대면 진료 이용 가능. 닥터나우와 나만의닥터가 시장을 양분하며, 각각 다른 전략으로 사용자를 확보 중.

- 비대면 진료 시범사업 지침 개정안 (2024.02.23 시행)
- 전화 진료 + 처방전 전송 + 약국 픽업/배달 모델 정착
- 헬스케어 앱 전반의 "슈퍼앱화" 추세 (진료 + 건강관리 + 리워드)
