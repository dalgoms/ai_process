# AI 건강 비서 — MVP 제안서

> 나만의닥터의 다음 성장 엔진: 일회성 방문자를 일상 사용자로 전환하는 AI 건강 동반자

---

## 왜 이 제안서를 만들었는가

나만의닥터는 **2만 개 이상의 건강 콘텐츠 페이지**로 검색 유입을 확보하는 데 이미 성공했습니다. 이 강력한 유입 엔진 위에 **일상 참여 레이어**를 얹으면, 콘텐츠를 읽으러 온 사용자가 매일 돌아오는 사용자가 될 수 있습니다.

AI 건강 비서는 이 가능성을 실현하기 위해 설계되었습니다.

## AI 건강 비서란?

AI 건강 비서는 사용자의 일상에 함께하는 AI 건강 어시스턴트입니다. 아플 때만 쓰는 앱이 아니라, 매일 함께하는 건강 파트너입니다.

사용자의 건강 행동을 분석하여 **매일 건강 인사이트**, **스마트 알림**, **건강 타임라인**을 제공합니다. 사용자가 자신의 건강 데이터를 보고, 이해하고, 행동할 수 있도록 돕습니다.

## 핵심 컨셉: 일상 건강 동반자

대부분의 헬스케어 앱은 특정 기능 중심으로 설계되어 있습니다 — 예약, 증상 확인, 약 복용 기록 등. 사용자는 문제가 생겼을 때 앱을 열고, 문제가 해결되면 잊어버립니다.

AI 건강 비서는 이 모델을 뒤집습니다. 사용자가 앱을 찾아오길 기다리는 대신, **앱이 사용자를 찾아갑니다** — 매일 아침과 저녁, 개인화된 건강 메시지와 함께.

## 나만의닥터에게 이것이 의미하는 것

```
현재:    SEO 유입  →  콘텐츠 열람  →  이탈 (일회성)
제안:    SEO 유입  →  콘텐츠 열람  →  AI 건강 비서 온보딩  →  일상 참여  →  비대면 진료/약 배달 전환
```

AI 건강 비서가 **유입과 수익 사이의 다리**가 됩니다.

## 주요 기능 (MVP)

| 기능 | 설명 | 나만의닥터 연결 |
|---|---|---|
| **스마트 건강 알림** | 수면, 걸음 수, 행동 패턴에 대한 AI 기반 일일 인사이트 | 이상 징후 감지 시 비대면 진료 연결 |
| **건강 챗봇** | 증상, 약물, 건강 조언에 대해 자연어로 질문 | 심화 상담 필요 시 의사 연결 |
| **건강 타임라인** | 모든 건강 활동과 데이터를 달력 형태로 조회 | 진료 이력 + 처방 이력 통합 |
