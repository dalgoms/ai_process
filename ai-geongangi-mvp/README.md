# AI 건강 비서 — MVP 제안서

> AI 기반 일상 건강 동반자.

## AI 건강 비서란?

AI 건강 비서는 사용자의 일상에 함께하는 AI 건강 어시스턴트입니다. 아플 때만 쓰는 앱이 아니라, 매일 함께하는 건강 파트너입니다.

사용자의 건강 행동을 분석하여 **매일 건강 인사이트**, **스마트 알림**, **건강 타임라인**을 제공합니다. 사용자가 자신의 건강 데이터를 보고, 이해하고, 행동할 수 있도록 돕습니다.

## 핵심 컨셉: 일상 건강 동반자

대부분의 헬스케어 앱은 특정 기능 중심으로 설계되어 있습니다 — 예약, 증상 확인, 약 복용 기록 등. 사용자는 문제가 생겼을 때 앱을 열고, 문제가 해결되면 잊어버립니다.

AI 건강 비서는 이 모델을 뒤집습니다. 사용자가 앱을 찾아오길 기다리는 대신, **앱이 사용자를 찾아갑니다** — 매일 아침과 저녁, 개인화된 건강 메시지와 함께.

## 주요 기능 (MVP)

| 기능 | 설명 |
|---|---|
| **스마트 건강 알림** | 수면, 걸음 수, 행동 패턴에 대한 AI 기반 일일 인사이트 |
| **건강 챗봇** | 증상, 약물, 건강 조언에 대해 자연어로 질문 |
| **건강 타임라인** | 모든 건강 활동과 데이터를 달력 형태로 조회 |
