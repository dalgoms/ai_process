# 솔루션 — AI 건강이

## 도구에서 동반자로

AI 건강이는 헬스케어 앱의 역할을 **수동적인 도구**에서 **능동적인 일상 동반자**로 전환하는 AI 건강 어시스턴트입니다.

사용자가 앱을 열기를 기다리는 대신, AI 건강이가 먼저 다가갑니다 — 사용자의 데이터, 행동, 패턴을 기반으로 개인화된 건강 인사이트를 전달합니다.

## 작동 방식

```
사용자 건강 데이터  →  AI 분석  →  일일 인사이트  →  사용자 행동
```

1. **수집** — 연동된 소스(웨어러블, 수동 기록, 앱 활동)에서 건강 신호를 수집합니다.
2. **분석** — AI가 수면, 활동, 약물, 바이탈, 식단 등의 패턴을 처리합니다.
3. **전달** — 개인화된 인사이트가 적절한 시점(아침, 저녁)에 사용자에게 전달됩니다.
4. **참여** — 사용자는 건강 타임라인을 통해 자신의 건강 이야기가 펼쳐지는 것을 확인합니다.

## 기존 앱과의 차이

| 기존 헬스케어 앱 | AI 건강이 |
|---|---|
| 아플 때 사용자가 앱을 연다 | 매일 앱이 사용자를 찾아간다 |
| 기능 중심 네비게이션 | 인사이트 중심 경험 |
| 정적 대시보드 | 살아있는 건강 타임라인 |
| 일반적인 푸시 알림 | AI 개인화 건강 넛지 |
| 데이터가 방치된다 | 데이터가 매일 가치를 만든다 |

## 핵심 아이디어

> "건강한 날에도 건강은 멈추지 않습니다. 건강 앱도 마찬가지여야 합니다."

AI 건강이는 건강 데이터를 **매일 유용하게** 만듭니다 — 아픈 날만이 아니라. 흩어진 데이터 포인트를 사용자가 보고, 이해하고, 행동할 수 있는 하나의 건강 이야기로 만들어줍니다.

---

*AI 건강이는 진단 도구가 아닙니다. 사용자가 자신의 건강에 관심을 갖고 참여할 수 있도록 돕는 일상 건강 동반자입니다.*
