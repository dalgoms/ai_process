# 건강 타임라인

## 컨셉

건강 타임라인은 흩어진 건강 데이터를 하나의 시간 순서 뷰로 변환합니다. 사용자가 취하는 모든 건강 행동 — 또는 자동으로 수집되는 데이터 — 이 개인 타임라인의 데이터 포인트가 됩니다.

**스스로 기록되는 건강 일기**라고 생각하면 됩니다.

## 건강 데이터가 타임라인이 되는 과정

```
데이터 소스            →    처리          →    타임라인 항목
─────────────────────      ──────────        ──────────────
웨어러블 (자동)             정규화             3/10 - 8,200보 걸음
수동 기록 (사용자)          분류               3/10 - 혈압약 복용
건강 앱 연동               중복 제거           3/10 - 체중: 72.3 kg
검사 결과 (업로드)          AI 보강            3/09 - 혈압: 128/82
```

## 데이터 유형

| 카테고리 | 예시 | 소스 |
|---|---|---|
| **약물** | 처방약 기록, 영양제 섭취, 용량 변경 | 수동 기록, 약국 연동 |
| **체중** | 일일 체중 측정, BMI 추세 | 수동 기록, 스마트 체중계 |
| **혈압** | 수축기/이완기 측정, 안정 시 심박수 | 수동 기록, 웨어러블 |
| **걸음 수 & 활동** | 일일 걸음 수, 운동 기록, 칼로리 소모 | 웨어러블, 스마트폰 센서 |
| **식단** | 식사 기록, 칼로리 섭취, 수분 섭취 | 수동 기록, 사진 기록 |
| **주사** | 인슐린, 예방접종, 알레르기 주사 | 수동 기록, 병원 기록 |

## 타임라인 뷰

타임라인은 각 날의 건강 이벤트 요약을 보여주는 **달력 기반 인터페이스**로 제공됩니다:

- **일간 뷰** — 하루 동안의 모든 데이터 포인트 상세 보기
- **주간 뷰** — 한 주의 종합 트렌드와 하이라이트
- **월간 뷰** — 전체적인 패턴, 연속 기록, 이상치

## AI 레이어

타임라인은 단순한 기록이 아닙니다. AI 건강이가 그 위에 지능 레이어를 추가합니다:

- **패턴 인식** — "혈압이 3일 연속 상승했습니다."
- **상관관계** — "8,000보 이상 걸은 날에 수면 질이 20% 향상됩니다."
- **공백 감지** — "5일간 체중을 기록하지 않으셨습니다. 업데이트하시겠어요?"
