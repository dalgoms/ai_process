# 나만의닥터 서비스 통합 시나리오

> AI 건강 비서가 나만의닥터 기존 서비스와 어떻게 연결되는가

---

## 통합 원칙

AI 건강 비서는 독립 제품이 아닙니다. 나만의닥터의 **기존 서비스 위에 올라가는 참여 레이어**입니다. 새로운 앱을 만드는 것이 아니라, 기존 앱의 빈 시간을 채우는 것입니다.

```
┌─────────────────────────────────────────────────┐
│                나만의닥터 앱                       │
│                                                  │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐       │
│  │ 비대면    │  │ 약 배달   │  │ 건강     │       │
│  │ 진료      │  │          │  │ 콘텐츠   │       │
│  └──────────┘  └──────────┘  └──────────┘       │
│                                                  │
│  ╔══════════════════════════════════════╗        │
│  ║       AI 건강 비서 (참여 레이어)       ║        │
│  ║  건강 타임라인 · 스마트 알림 · 챗봇    ║        │
│  ╚══════════════════════════════════════╝        │
│                                                  │
└─────────────────────────────────────────────────┘
```

## 통합 시나리오

### 시나리오 1: 비대면 진료 연결

사용자의 건강 데이터에서 이상 패턴을 감지하면, AI 건강 비서가 비대면 진료를 제안합니다.

```
건강 타임라인 데이터                   AI 판단                      액션
───────────────────                  ────────                    ──────
혈압 3일 연속 상승                    이상 패턴 감지               "혈압이 3일째 올라가고 있어요.
(128→135→142)                       위험도: 중간                   의사와 상담해볼까요?"
                                                                [비대면 진료 예약하기]
```

**기존**: 사용자가 불편을 느끼고 → 스스로 앱을 열어 진료 예약
**제안**: AI가 데이터 기반으로 선제 제안 → 원탭 진료 예약

### 시나리오 2: 처방약 리필 자동화

건강 타임라인에 기록된 약물 복용 데이터를 기반으로, 리필 시점을 예측하고 약 배달을 연결합니다.

```
약물 기록                           AI 예측                      액션
────────                           ────────                    ──────
메트포르민 30일분 처방               25일째 (5일 남음)             "처방약이 5일 뒤 소진 예정이에요.
매일 1회 복용 기록 중                리필 필요 예측                  리필 진료를 예약할까요?"
                                                                [리필 예약] [약 배달 주문]
```

### 시나리오 3: 건강 콘텐츠 개인화 추천

나만의닥터의 2만 개 healthLab 콘텐츠를 사용자의 건강 데이터와 매칭하여 추천합니다.

```
사용자 데이터                        매칭 로직                    추천
──────────                         ──────────                  ──────
최근 혈압 상승 트렌드               healthLab 콘텐츠 중           "고혈압 예방을 위한 생활 습관 가이드"
수면 부족 패턴                      관련 키워드 매칭              "수면의 질을 높이는 5가지 방법"
걸음 수 감소                                                    "사무실에서 할 수 있는 간단한 운동"
```

**기존**: 사용자가 검색해서 콘텐츠를 찾음 (SEO 의존)
**제안**: AI가 사용자에게 맞는 콘텐츠를 푸시 (개인화 추천)

### 시나리오 4: 진료 이력 통합

비대면 진료 이력, 처방 이력이 건강 타임라인에 자동 기록됩니다.

```
3월 10일  ─  비대면 진료 (내과) · 혈압 상담
3월 10일  ─  처방: 아모디핀 5mg, 30일분
3월 11일  ─  약 배달 수령
3월 12일  ─  약물 복용 기록 ✓
    ...
4월 5일   ─  AI 알림: "리필 5일 전입니다"
```

진료 → 처방 → 복용 → 리필까지 끊김 없는 건강 관리 루프가 완성됩니다.

### 시나리오 5: 챗봇 → 진료 에스컬레이션

건강 챗봇에서 증상 상담 중, AI가 의료 전문가 상담이 필요하다고 판단하면 비대면 진료로 자연스럽게 연결합니다.

```
사용자: "3일째 두통이 있고 어지러워요"

AI 건강 비서: "타임라인을 확인해보니, 최근 수면이
             평균 4.5시간으로 많이 부족하셨네요.
             수면 부족이 원인일 수 있지만,
             3일 이상 지속되는 두통은
             의사와 상담하시는 것을 권합니다."

             [비대면 진료 바로 예약]
             [증상 더 알아보기 → healthLab]
```

## 데이터 흐름 아키텍처

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  데이터 수집   │     │   AI 분석     │     │   서비스 연결  │
│              │     │              │     │              │
│ 웨어러블     │────→│ 패턴 인식    │────→│ 스마트 알림   │
│ 수동 기록    │     │ 이상 감지    │     │ 진료 제안     │
│ 진료 이력    │     │ 트렌드 분석  │     │ 약 리필       │
│ 처방 이력    │     │ 상관관계     │     │ 콘텐츠 추천   │
│ 약물 복용    │     │              │     │ 챗봇 응답     │
└──────────────┘     └──────────────┘     └──────────────┘
       │                                         │
       └──────────── 건강 타임라인 ────────────────┘
```

## 기존 기능에 미치는 영향

| 기존 기능 | AI 건강 비서 통합 전 | 통합 후 |
|---|---|---|
| 비대면 진료 | 사용자가 아플 때 직접 예약 | AI가 선제 제안 → 예약 전환율 증가 |
| 약 배달 | 처방 후 수동 주문 | 리필 시점 자동 알림 → 재주문율 증가 |
| 건강 콘텐츠 | SEO 유입 후 이탈 | 개인화 추천 → 콘텐츠 재소비 + 체류 시간 증가 |
| 앱 전체 | 아플 때만 방문 | 매일 아침/저녁 방문 → DAU 대폭 증가 |

## 수익 기여 모델

```
AI 건강 비서 (무료)
    │
    ├── 일상 참여 → DAU 증가 → 앱 가치 증가
    │
    ├── 이상 감지 → 비대면 진료 전환 → 진료 수수료
    │
    ├── 약물 리필 → 약 배달 전환 → 배달 수수료
    │
    ├── 건강 데이터 → 보험사/제약사 파트너십 → B2B 수익
    │
    └── 프리미엄 인사이트 → 구독 모델 (향후)
```

AI 건강 비서 자체는 무료로 제공하되, 기존 수익원(진료, 약 배달)의 전환율을 높이는 **성장 엔진** 역할을 합니다.
