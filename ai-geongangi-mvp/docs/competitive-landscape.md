# 경쟁 환경 분석

> 헬스케어 앱 시장에서 AI 건강 비서의 포지셔닝

---

## 시장 현황

한국 디지털 헬스케어 시장은 크게 **비대면 진료**, **건강 관리**, **보상형 건강** 세 축으로 나뉩니다. 각 축의 주요 플레이어는 명확한 강점과 한계를 갖고 있습니다.

## 주요 경쟁사 분석

| 서비스 | 핵심 가치 | 강점 | 약점 |
|---|---|---|---|
| **닥터나우** | 비대면 진료 + 약 배달 | 빠른 진료, 약 배달 인프라, 시장 1위 인지도 | 아프지 않으면 앱을 열 이유 없음, 일상 참여 제로 |
| **굿닥** | 병원 예약 + 접수 | 오프라인 병원과의 강한 연결, 예약 편의성 | 예약 이후 재방문 동기 부재, 리텐션 구조 없음 |
| **캐시워크** | 걸음 수 보상 | 높은 DAU (보상 동기), 가벼운 진입 장벽 | 건강 행동 변화가 아닌 보상에 의존, 의료 서비스 연결 약함 |
| **삼성 헬스** | 기기 연동 건강 관리 | 웨어러블 데이터 자동 수집, 방대한 사용자 기반 | 데이터 대시보드에 그침, 행동 유도 없음, 의료 서비스 미연결 |
| **나만의닥터** | 비대면 진료 + 건강 콘텐츠 | SEO 기반 대량 유입, 2만+ 콘텐츠 페이지 | 콘텐츠 이후 이탈, 일상 참여 장치 부재 |

## 참여 격차 (Engagement Gap)

모든 경쟁사에 공통된 구조적 문제가 있습니다:

```
┌─────────────────────────────────────────────────┐
│           헬스케어 앱의 참여 스펙트럼              │
│                                                  │
│  아플 때만                    매일                │
│  ←──────────────────────────────────────→        │
│                                                  │
│  닥터나우  굿닥    나만의닥터    캐시워크  삼성헬스 │
│  ●        ●       ●            ●        ●       │
│                                                  │
│                        AI 건강 비서 ◆             │
│                        (목표 포지션)              │
└─────────────────────────────────────────────────┘
```

- **좌측 (아플 때만)**: 진료/예약 앱 — 문제 해결 후 이탈
- **우측 (매일)**: 보상/트래킹 앱 — 사용하지만 의료 서비스와 단절
- **AI 건강 비서**: 일상 참여와 의료 서비스를 연결하는 **유일한 포지션**

## 나만의닥터 + AI 건강 비서의 차별점

| 차원 | 경쟁사 | 나만의닥터 + AI 건강 비서 |
|---|---|---|
| **유입** | 앱스토어 검색, 광고 | SEO 2만 페이지 + 퍼포먼스 광고 |
| **일상 참여** | 없음 또는 보상 의존 | AI 개인화 아침/저녁 건강 루프 |
| **데이터 활용** | 대시보드에 방치 | 매일 인사이트로 전환 |
| **의료 연결** | 진료만 가능 | 일상 건강 관리 → 이상 징후 시 자동 진료 연결 |
| **리텐션 구조** | 없음 | 건강 타임라인 + 챗봇 + CRM 캠페인 |

## 핵심 포지셔닝

> **"매일 쓰는 건강 비서가 있는 유일한 비대면 진료 플랫폼"**

나만의닥터만이 가진 자산:
1. **2만 개 SEO 콘텐츠** → 유입 엔진 이미 확보
2. **비대면 진료 인프라** → 건강 이상 감지 시 즉시 의사 연결 가능
3. **약 배달 시스템** → 처방 후 배달까지 원스톱

이 위에 AI 건강 비서가 올라가면, 경쟁사는 콘텐츠(유입) + 일상 참여(리텐션) + 진료(수익)를 모두 갖춘 구조를 복제하기 어렵습니다.

## SWOT 요약

| | 긍정 | 부정 |
|---|---|---|
| **내부** | **S** — 콘텐츠 유입 엔진, 진료 인프라, 약 배달 | **W** — 리텐션 구조 부재, 퍼포먼스 랜딩 미비 |
| **외부** | **O** — AI 건강 관리 수요 증가, 만성질환자 증가 | **T** — 닥터나우 시장 점유, 규제 불확실성 |

AI 건강 비서는 **W(약점)를 직접 해결**하면서 **O(기회)를 선점**하는 전략입니다.
