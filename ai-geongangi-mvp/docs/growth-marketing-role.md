# Growth Marketing 실행 계획

> 합류 후 90일 — AI 건강 비서를 중심으로 나만의닥터의 성장 구조를 설계합니다

---

## 왜 Growth Marketing인가

AI 건강 비서는 좋은 제품 아이디어만으로는 성공하지 못합니다. **유입 → 온보딩 → 참여 → 전환**의 전 과정을 설계하고, 데이터로 검증하고, 반복 최적화하는 실행력이 필요합니다.

10년간 SORIZAVA에서 **3.16M 세션 / +1,078% 전환**을 만들어낸 경험은 바로 이 실행력입니다.

## 90일 로드맵

### Phase 1: Day 1–30 — 진단과 기반 구축

```
┌─────────────────────────────────────────────────┐
│  Week 1-2: 현황 진단                              │
│  ─────────────────                               │
│  · GA4 / CRM 데이터 전수 분석                     │
│  · 현재 유입-전환 퍼널 맵핑                        │
│  · 이탈 구간 및 전환 병목 식별                     │
│  · 경쟁사 대비 포지셔닝 업데이트                   │
│                                                  │
│  Week 3-4: 기반 설계                              │
│  ─────────────────                               │
│  · 퍼포먼스 랜딩 페이지 설계 (현재 부재)           │
│  · UTM 체계 설계 및 구축                          │
│  · AI 건강 비서 온보딩 퍼널 설계                   │
│  · CRM 이벤트 택소노미 정의                       │
└─────────────────────────────────────────────────┘
```

**산출물:**
- 현황 진단 리포트 (SORIZAVA 성과 리포트 수준)
- UTM 추적 체계 (SORIZAVA에서 구축한 엔드투엔드 시스템 적용)
- 퍼포먼스 랜딩 페이지 v1

### Phase 2: Day 31–60 — MVP 런칭 & 초기 최적화

```
┌─────────────────────────────────────────────────┐

│  ─────────────────                               │

│  · 챗봇 → 진료 전환 경로 분석                     │
```
