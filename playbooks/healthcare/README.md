# AI Healthcare Automation Playbook

헬스케어 플랫폼에서 AI 기반 업무 자동화를 적용하는 전략과 구조를 정리한 플레이북입니다.

---

## 대상 서비스

- 비대면 진료 플랫폼 (나만의닥터 등)
- 디지털 헬스케어 서비스
- 병원 CRM / 파트너 관리 시스템
- 건강관리 앱

---

## 구성

### [Business Case](../../docs/business-case-healthcare.md)

헬스케어 플랫폼에 AI 자동화를 적용하는 구체적 사례.

- B2B 파트너(병원/약국) 영업 자동화
- B2C 환자 문의 AI 분류 및 라우팅
- 콘텐츠 마케팅 자동 생산
- 내부 개발 버그 수정 자동화

### [Growth Loop](./growth-loop.md)

AARRR 프레임워크 기반 사용자 성장 루프.

- Acquisition: 검색 유입, 건강 콘텐츠, 광고
- Activation: 첫 진료 경험, 약 배송
- Retention: 복약 알림, 재진료 리마인더
- Referral: 가족 추천, 리뷰, 커뮤니티

### [Automation Architecture](./automation-architecture.md)

자동화 기술 구조도.

- 리드 캡처 → CRM → 알림 파이프라인
- 고객 문의 AI 자동 분류
- 콘텐츠 생산 AI 파이프라인
- 코드 자동 수정 파이프라인

### [Data Layer](./data-layer.md)

데이터 구조와 AI 활용 설계.

- Patient Data (진료, 처방, 배송)
- Partner Data (병원, 진료과목, 성과)
- Marketing Data (유입, 전환, 콘텐츠)
- AI 활용 시나리오

## 핵심 목표

AI 자동화를 통해 달성하는 것:

| 목표 | 지표 |
|------|------|
| 운영 효율 향상 | 수동 작업 80% 감소 |
| 리드 응답 시간 단축 | 24시간 → 즉시 (0분) |
| 콘텐츠 생산 확대 | 월 4건 → 월 20건 |
| 파트너 이탈 방지 | 팔로업 누락 0건 |
| 개발 속도 향상 | 핫픽스 시간 80% 단축 |

---

## 활용 방법

1. Business Case로 자동화 적용 범위 파악
2. Growth Loop로 마케팅/성장 전략 수립
3. Automation Architecture로 기술 구현
4. Data Layer로 데이터 기반 AI 활용 설계
